Quick Start üöÄ

import 'package:murmuration/murmuration.dart';
import 'package:google_generative_ai/google_generative_ai.dart';

void main() async {
    final config = MurmurationConfig(
        apiKey: 'your-api-key',
        model: 'gemini-1.5-flash-latest',
        debug: false,
        stream: false,
        logger: MurmurationLogger()
    );

    final murmur = Murmuration(config);

    final result = await murmur.run(
        input: "Hello!",
        agentInstructions: {
            'role': 'You are a helpful assistant.'
        }
    );

    print(result.output);
}

Core Concepts üß†

Agents are the fundamental building blocks in Murmuration, powered by Google's GenerativeModel. They support:

  • Instructions defining behavior
  • Tools and functions
  • State management
  • Progress reporting

Best Practices üèÜ

  • Use immutable state operations.
  • Test function call string parsing thoroughly.
